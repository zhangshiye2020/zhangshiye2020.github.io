nvm

nodejs的版本管理工具，为解决nodejs更新速度实在太快给出的解决方案，以下为nvm windows常用命令

nvm list [available]

列出nodejs版本

  • available:
    • available 列出所有可选的nodejs版本号
    • installed 列出所有已安装的nodejs版本

nvm install <version> [arch]

安装指定版本号的nodejs

  • version: nodejs版本号，可根据上面nvm list获取
  • arch: 计算机架构，32还是64

nvm use [version] [arch]

指定使用的nodejs版本号

  • version: 已安装的nodejs版本号
  • arch: 计算机架构
